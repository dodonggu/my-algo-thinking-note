# 🧠 My-Algo-Thinking-Note

> **"무작정 코드를 작성하기보다, 논리를 먼저 세우는 연습을 기록합니다."**

단순히 문제를 해결하는 것에 그치지 않고, **0단계부터 4단계까지의 설계 프로세스**를 거쳐 최적의 로직을 도출하는 과정을 아카이브합니다.

## 🚀 문제 해결 5단계 프로세스

모든 문제는 아래의 템플릿에 따라 분석하고 설계합니다. 

1. 0. 문제 분석: 시간 복잡도 계산 및 제약 조건 파악 


2. 1. 전략 수립: 알고리즘 기법 선정 (DFS, BFS, DP 등) 


3. 2. 상태 정의: 데이터 구조화 및 변수 의미 정의 


4. 3. 핵심 로직: 종료 조건과 진행 조건 명시 


5. 4. 시뮬레이션: 손으로 직접 데이터의 변화를 검증 



---

## 📂 폴더 구조

* **`DP/`**: 동적 계획법 관련 설계 노트 및 코드
* **`DFS_BFS/`**: 탐색 관련 설계 노트 및 코드
* **`Graph/`**: 그래프 알고리즘 관련 정리
